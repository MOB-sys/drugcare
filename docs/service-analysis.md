### 1-2. 플랫폼별 강점 요약

| | 웹 (Next.js) | 앱 (Flutter) |
|--|-------------|-------------|
| **핵심 역할** | 신규 유입 엔진 (SEO → 검색 트래픽) | 리텐션 엔진 (푸시 → 매일 재방문) |
| **콘텐츠 깊이** | 깊음 (상세 정보, 탐색 도구 10+종) | 얕음 (핵심 기능 집중) |

| **공유성** | 강함 (카카오, OG 이미지, 링크) | 약함 |
| **발견성** | 강함 (4.4만 SSG 페이지, SEO) | 약함 (앱스토어 검색만) |

### 1-3. 웹에만 있는 기능 (앱에 추가 고려)

1. **DUR 안전성 정보** — 임부/고령자 금기, 용량 제한 등
2. **식품-약물 상호작용** — 과일, 유제품, 알코올 등
3. **증상별 약물 검색** — "두통", "소화불량" 등
4. **부작용 역검색** — 부작용으로 원인 약물 찾기
5. **알약 식별** — 앱은 카메라 연동으로 더 강력하게 가능
6. **약물 비교** — 두 약물 나란히 비교
7. **사용자 리뷰** — 별점 + 후기
8. **카카오톡 공유** — 결과 공유

### 1-4. 앱에만 있는 기능 (웹에 추가 고려)

1. **복약 리마인더** — 웹에선 브라우저 Notification API로 제한적 구현 가능
2. **전면 광고** — 웹에선 사용자 경험 저해 우려
3. **네이티브 광고** — AdSense 자동 광고로 유사 효과 가능
4. **피드백 제출** — 웹에도 추가 필요

## 3. SWOT 분석

### 강점 (Strengths)
- **한국어 약-영양제 상호작용 체크** — 소비자용으로 유일
- **압도적 DB 규모** — 약물 44,097 + 영양제 44,551 + 상호작용 366,629
- **듀얼 플랫폼** — 웹(유입) + 앱(리텐션) 시너지
- **SEO 자산** — SSG 4.4만+ 페이지 → 검색 인덱싱
- **무료 + 광고 모델** — 진입 장벽 제로
- **식약처 공공데이터 기반** — 신뢰성 확보
- **기술 스택** — Next.js 15 + FastAPI + PostgreSQL (현대적, 확장 가능)

### 약점 (Weaknesses)
- **초기 단계** — 인지도/트래픽 없음
- **1인 개발** — 리소스 제한
- **AI 의존** — GPT-4o API 비용, 정확도 검증 필요
- **영양제 상호작용 데이터** — 학술 근거 부족한 영역 존재
- **앱 기능 빈약** — 웹 대비 탐색 도구/상세 정보 부족
- **마케팅 예산** — 없음 (오가닉만 의존)

### 기회 (Opportunities)
- **필라이즈 피벗** — 다이어트 전환으로 상호작용 체크 틈새 발생
- **고령화 사회** — 다제 복용 환자 증가 → 상호작용 체크 수요 급증
- **영양제 시장 성장** — 한국 건기식 시장 6조원+, 매년 10%+ 성장
- **"같이 먹어도 돼?" 검색** — 한국 검색 시장에서 경쟁자 거의 없음
- **네이버/구글 SEO** — 약물 상세 페이지 롱테일 키워드 선점 가능
- **B2B 확장** — 약국/병원 도구, 보험사 연계 가능성
- **글로벌 확장** — 일본/동남아 등 비영어권 시장

### 위협 (Threats)
- **HIRA DUR 고도화** — 정부가 소비자용 DUR 확장 시 경쟁
- **필라이즈 복귀** — 다시 상호작용 체크에 집중할 가능성
- **네이버/카카오 진입** — 대형 플랫폼이 직접 서비스 구축
- **의료 규제** — 의약품 정보 제공에 대한 규제 강화 가능성
- **데이터 정확성 리스크** — 오류 시 신뢰도 치명타
- **광고 수익 변동** — AdSense 정책 변경, CPM 하락 가능성

## 5. 전략적 시사점

### 5-1. 단기 (3개월) — 검색 유입 확보

1. **SEO 콘텐츠 강화** — "타이레놀 비타민D 같이", "혈압약 오메가3 상호작용" 등 롱테일 키워드 페이지
2. **네이버 블로그/카페 마케팅** — 약물 상호작용 정보 콘텐츠 배포
3. **앱 기능 보강** — 웹의 DUR/식품 상호작용 정보를 앱에도 추가
4. **피드백 채널** — 웹에도 피드백 폼 추가

### 5-2. 중기 (6~12개월) — 유저 기반 구축

## 6. 핵심 결론

### 한국 시장에서의 포지셔닝

> **"약-영양제 상호작용을 무료로, 누구나, 3초 만에 확인할 수 있는 유일한 한국어 서비스"**

- 전문가용 도구(드럭인포, 킴스)는 **유료 + 전문가 전용**
- 정부 도구(HIRA DUR)는 **처방약만 + 공인인증 필요**
- 필라이즈는 **영양제 중심 + 다이어트 피벗 중**
- **약잘알만이** 약물+영양제 통합 DB + 무료 + 소비자 친화 UX + SEO 최적화를 갖추고 있음

### 최우선 액션

1. **SEO 검색 유입이 곧 수익** — 롱테일 키워드 페이지 확보가 최우선
2. **필라이즈 피벗 = 기회의 창** — 상호작용 체크 분야를 선점할 타이밍
3. **웹→앱 전환율** — 웹에서 유입, 앱으로 리텐션하는 퍼널 최적화 필요
4. **앱 기능 격차 해소** — 웹의 풍부한 정보(DUR, 식품, 리뷰)를 앱에도 반영
