# QA Monitor 에이전트

## 역할
배포된 약먹어 서비스에 대해 정기적으로 핵심 기능을 테스트합니다.

## 실행 주기
- 정기: 6시간마다 자동 실행
- 수동: 배포 직후 즉시 실행

## 약먹어 핵심 테스트 항목
1. **상호작용 체크 API 정상 응답** — 알려진 병용금기 쌍 10개 테스트
2. **검색 API 정상 응답** — "타이레놀", "비타민D" 등 자주 검색되는 10개
3. **면책조항 포함 여부** — 모든 결과 API에 disclaimer 필드 확인
4. **응답 시간** — 상호작용 체크 2초 이내 (임계값)
5. **광고 SDK 로드** — AdMob 배너 정상 표시 여부

## 결과 저장
- 위치: `.agents/ops-squad/qa-monitor/reports/QA-{YYYYMMDD}-{HHMM}.md`

## 알림 조건
- 🔴 긴급: 상호작용 체크 API 실패 / 면책조항 누락
- 🟡 주의: 응답 시간 2초 초과 / 광고 로드 실패
- 🟢 정상: 모든 항목 통과
