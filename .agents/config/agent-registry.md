## 약먹어 특수 에이전트: Data Engineer

이 프로젝트의 핵심은 **약물 데이터의 정확성**입니다. 일반 프로젝트와 달리 Data Engineer 에이전트가 추가되어 있으며, 다음을 전담합니다:

1. 식약처 공공데이터 API 수집 스크립트 개발
2. DUR 병용금기 데이터 파싱 및 정규화
3. 한국 영양제 성분 매핑 DB 구축
4. 상호작용 체크 엔진 핵심 로직 개발
5. 데이터 품질 검증 파이프라인

**⚠️ 중요:** 약물 데이터 관련 코드는 반드시 Data Engineer가 담당하며, Backend 에이전트는 API 레이어만 담당합니다.
