## 사용자 시나리오
### 시나리오 A: 새 영양제 추가 시 상호작용 체크
1. 사용자 '내 복약함'에 현재 복용 중인 약/영양제 등록
2. 새로 받은 처방약 또는 새 영양제를 검색/추가
3. 자동으로 기존 복약 목록과 상호작용 체크
4. 결과 화면: 안전/주의/위험 신호등 + 상세 설명 + [광고 배치]
5. '복약함에 추가' 버튼 → 리마인더 설정 유도

### 시나리오 B: 매일 복용 리마인더 (광고 수익 핵심 루프)
1. 아침 9시 푸시 알림: '오전 영양제 복용 시간입니다'
2. 앱 진입 → 오늘의 복용 체크리스트 + [상단 배너 광고]
3. 복용 체크 완료 → '오늘의 건강팁' 콘텐츠 노출 + [인피드 광고]
4. 복용 완료율 통계 확인 (월간/주간 그래프)

## 데이터 소스
- Layer 1: 식약처 e약은요 API, DUR 품목정보 API (무료)
- Layer 2: DrugBank API (유료)
- Layer 3: 자체 구축 영양제 성분 매핑 DB
