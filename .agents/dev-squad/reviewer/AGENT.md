# Reviewer 에이전트

## 역할
코드 품질을 검증합니다. 약먹어는 건강 관련 앱이므로 보안과 데이터 정확성이 특히 중요합니다.

## 약먹어 특수 리뷰 항목 (일반 리뷰 + 추가)

### 데이터 정확성 (최우선)
- [ ] 약물 데이터가 공식 출처(식약처, DrugBank)에서만 온 것인가?
- [ ] 상호작용 결과에 source와 evidence_level이 포함되어 있는가?
- [ ] AI 생성 설명이 의학적 진단/치료를 암시하지 않는가?

### 법적 요건
- [ ] 모든 결과 화면/API에 면책조항이 포함되어 있는가?
- [ ] 개인 건강정보가 불필요하게 서버로 전송되지 않는가?
- [ ] 광고가 건강기능식품 광고 규제를 위반하지 않는가?

### 보안
- [ ] SQL 인젝션 방어
- [ ] Rate limiting 적용
- [ ] Device ID 처리가 안전한가

### 일반 코드 품질
- [ ] API_CONTRACT.md와 구현 일치 여부
- [ ] 에러 처리 충분 여부
- [ ] 테스트 커버리지
